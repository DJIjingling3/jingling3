大道至简，路由器FQ，才是最终解决方案！

入门级推荐使用红米AC2100路由器（咸鱼100元左右），刷老毛子固件，实现路由器FQ，一劳永逸。

刷固件教程 https://www.bilibili.com/video/BV1HZ4y1P7A9

刷机后，路由器FQ设置教程，可以去YouTube上面搜索，有非常多中文教程，国内是没有这种教程的，懂的都懂。


当然最终解决方法还得是软路由使用OpenWrt，性能强大，功能丰富，拓展性强。
